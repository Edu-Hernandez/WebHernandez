Proyecto de Desarrollo Web

Implementaciòn de una pàgina web para el proceso de ventas 2022 - Actualidad

Como desarrollador en un proyecto académico, participé en la creación de un sitio web interactivo diseñado para optimizar el proceso de ventas. ----- 2024

Formé parte del desarrollo de una aplicación móvil utilizando Flutter para un restaurante, diseñada para facilitar el adelanto de sueldo a los empleados mediante el uso de la cámara del dispositivo. La información se guarda en Firebase
